## 🏗️ About the Engine
BrickHouse is a custom-built, lightweight execution environment designed for stability and protection. By separating the engine logic from the Python environment, we ensure a tiny footprint (<1MB) that runs efficiently on legacy hardware.

### The Three Pillars:
* **High-Stability Persistence:** Uses a **Resilient Loop** architecture—the shell stays alive even if your script crashes.
* **The Multi-Layer Shield:** Includes the **Audit Gate** (intercepts unauthorized system calls) and **The Shroud** (masks CPU spikes).
* **Integrity Verification:** Anchored to the `sequenceman.rar` file to ensure the environment is genuine.
